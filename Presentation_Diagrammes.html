GeoTrack — Rapport PFE2

1 Description Générale du Projet

GeoTrack est une plateforme web SaaS complète de géolocalisation GPS en temps réel, conçue pour le marché tunisien. Elle permet aux entreprises de suivre leurs véhicules, personnes, animaux et objets de valeur via des traceurs GPS professionnels et une interface web moderne.

Objectifs du projet

  • Développer une plateforme multi-tenant permettant à chaque entreprise de gérer ses propres appareils GPS
  • Offrir un suivi en temps réel avec mise à jour toutes les 5 secondes via WebSocket
  • Implémenter un système de géofencing avancé (zones circulaires et polygonales)
  • Fournir un système d'alertes automatiques (vitesse, batterie, hors ligne, géofence)
  • Intégrer un assistant IA commercial et un chatbot dans le tableau de bord
  • Mettre en place un workflow complet de commande, facturation et support client

Technologies utilisées

CoucheTechnologies
FrontendReact 18, TypeScript, Vite, TailwindCSS, Shadcn/ui, Zustand, Leaflet, Recharts
BackendNode.js, Express.js, Knex.js, WebSocket (ws), bcrypt
Base de donnéesPostgreSQL (migré depuis MongoDB)
Services externesMailjet (email), n8n + Ollama (IA), Cloudflare Tunnel

Communication temps réel

Le serveur WebSocket diffuse les mises à jour GPS à tous les clients connectés, avec filtrage par rôle et entreprise. Les administrateurs reçoivent toutes les données, tandis que les opérateurs ne voient que les appareils de leur entreprise.

11 Conclusion

Le projet GeoTrack représente une solution complète et fonctionnelle de géolocalisation GPS multi-entreprises, développée avec des technologies modernes et adaptée au marché tunisien.

Réalisations principales

  • 19 pages frontend avec interface responsive et design premium (Shadcn/ui + TailwindCSS)
  • 15 routeurs API couvrant tous les domaines métier (60+ endpoints)
  • 14 tables PostgreSQL avec relations normalisées et indexes optimisés
  • Temps réel complet via WebSocket avec filtrage par rôle et entreprise
  • Moteur de simulation GPS réaliste avec 11 profils comportementaux
  • Géofencing avancé avec détection de transitions (entrée/sortie)
  • IA intégrée via n8n + Ollama pour assistance commerciale
  • Pipeline e-commerce complet : commande → facturation → activation
  • Sécurité robuste : RBAC 3 rôles, bcrypt, OTP, isolation des données

Perspectives d'évolution

  • Ajout de middleware JWT pour sécuriser toutes les routes API
  • Implémentation de tests unitaires et d'intégration
  • Conteneurisation avec Docker + pipeline CI/CD
  • Documentation API avec Swagger/OpenAPI
  • Application mobile native (React Native)
  • Intégration de protocoles GPS hardware (GT06, H02, T55)
  • Tableau de bord analytique avancé avec BI
